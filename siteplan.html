Website Planning Document

Site Name

Radar Inmobiliario México – Este nombre refleja el objetivo del sitio de mostrar análisis de datos inmobiliarios por colonia en México.

Site Purpose

Este sitio proporcionará reportes por colonia con datos de precios promedio, rangos de precios, plusvalía y tendencias. Estará dirigido a compradores, inversionistas y agentes inmobiliarios.

Scenarios

  • ¿Cuáles son las colonias con mayor plusvalía en Ciudad Juárez?
  • ¿Dónde puedo encontrar un reporte confiable con precios promedio de casas en venta?

Color Schema

  • Color principal: #2c3e50 (Encabezados y navegación)
  • Color secundario: #18bc9c (Botones, enlaces y elementos destacados)

Typography

  • Montserrat: Para encabezados (Google Fonts)
  • Open Sans: Para texto del cuerpo
